# Logo Generation Prompt

Create an app icon / logo mark for a local-first desktop app where a human moderates one shared chat between multiple AI coding agents.

The mark should communicate:
- one shared conversation,
- multiple provider-neutral participants,
- local control and user moderation,
- calm professional developer-tool energy.

Visual direction:
- abstract vector-style logo,
- simple geometric forms,
- a central speech-bubble or conversation shape,
- three subtle participant nodes feeding into the same conversation,
- flat or lightly dimensional design,
- strong silhouette that works at 16px and 32px,
- transparent or plain dark background,
- palette: electric blue, teal, violet, and one small warm accent.

Avoid:
- robots,
- human silhouettes,
- checkmarks,
- approval badges,
- org charts,
- orbit diagrams,
- meeting tables,
- cute mascots,
- complex dotted paths,
- neon glow,
- busy UI screenshots,
- text, letters, or words.

Output should feel like a serious AI developer tool, not a generic chatbot product.
